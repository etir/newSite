    Example: The user search for his degree.
    • The user opens the browser.
    • The user enters a familiar site that contains articles about universities in Israel.
    • The user searches for the article that is relevant to him.
    • The user reads the article he wanted.
    • The user opens the browser
    • The user enters his favorite search engine.
    • The user searches by engine for the desired article.
    • The user receives a list of sites that have articles about universities in Israel (and Ben-Gurion University).
    • The user chooses the site he wanted, he can read the content, comment on the forums, edit the information (if it is a site like Wikipedia) and more.
    • The user opens the browser.
    • The user writes his query on Google.
    • The user receives results from Google based on AI techniques and using databases based about users posts.
    • There are systems that collect information on the Internet that millions of users have published, analyzed and presented to the user smart information about choosing a place of study and a profession.
